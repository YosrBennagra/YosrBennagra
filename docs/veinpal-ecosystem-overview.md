# Veinpal Ecosystem Overview

Veinpal is a centralized platform that hosts a collection of software products including desktop applications, web applications, utility tools, and games. It acts as the main hub where all projects are organized and distributed.

## Core Platform

### [veinpal.com](https://www.veinpal.com/)

The main platform that aggregates and presents all products. It serves as the entry point for users to discover, access, and download applications.

## Desktop Application

### OverlayOS

A Windows desktop host application designed to run modular tools.

- Acts as a platform inside Windows
- Supports independently developed tools (modules/plugins)
- Tools can be added or removed dynamically

## Internal System

### OverlayOS ToolsPortal

An internal admin platform used to manage the OverlayOS ecosystem.

- Add/remove tools
- Control tool availability
- Manage the tool marketplace inside OverlayOS

## Web Applications

### Universal AI Library

A web platform that provides AI agent skills and capabilities, similar to tools like Copilot or Claude.

### ShipKit

A system and methodology used to build and ship products efficiently. It serves as a standardized framework reused across all projects.

## Utility Websites

### GetIconFav

A web tool for generating favicons from images.

### AllYourTypes

A file conversion platform that supports images, videos, and other formats.

## Games

### Typew (temporary name)

A website that hosts mini web-based games.

## Overall Structure

The ecosystem is composed of:

- A central platform (veinpal.com)
- A desktop system (OverlayOS + tools)
- A set of web applications (AI, ShipKit)
- Utility tools (favicon generator, file converter)
- A games platform
- An internal admin system (ToolsPortal)

## Concept

Veinpal is a multi-product ecosystem combining:

- A desktop platform (OverlayOS)
- A collection of web-based tools and applications
- A structured system to build, manage, and distribute software products
